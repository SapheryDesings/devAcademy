## 1. Introducción a Node.js
- **¿Qué es Node.js?**
  - Entorno de ejecución basado en JavaScript.
  - Ejecuta JavaScript fuera del navegador.
  - Basado en el motor V8 de Google Chrome.
  - Asíncrono y orientado a eventos.
  - Ideal para aplicaciones en tiempo real.

## 2. Instalación y Configuración
- Instalación de Node.js (Windows, macOS, Linux).
- Comprobación de la instalación: `node -v` y `npm -v`.
- Configuración del entorno de desarrollo.

## 3. Módulos en Node.js
- **Módulos Nativos**:
  - `fs` (sistema de archivos).
  - `http` (servidor HTTP).
  - `path` (gestión de rutas).
  - `events` (emisión de eventos).
  - `url` (manejo de URLs).
- **Módulos de Terceros**:
  - Instalación con `npm` (Node Package Manager).
  - Ejemplos populares: Express, Lodash, Axios.
- **Creación de Módulos Propios**.

## 4. Event Loop y Asincronía
- **¿Qué es el Event Loop?**
  - Concepto de ciclo de eventos y no bloqueante.
  - Callback functions (funciones de retorno).
- **Manejo de operaciones asíncronas**:
  - Callbacks.
  - Promesas (`Promise`).
  - `async/await`.
- **Flujo de ejecución en Node.js**.

## 5. Manejo de Paquetes con npm (Node Package Manager)
- Instalación de paquetes con `npm install`.
- Creación y gestión de archivos `package.json`.
- Dependencias y dependencias de desarrollo.
- Scripts en `package.json`.

## 6. Desarrollo de Servidores HTTP
- **Servidor básico con Node.js**.
  - Crear un servidor HTTP con el módulo `http`.
  - Manejo de peticiones y respuestas.
- **Uso de Express.js**.
  - Framework minimalista para servidores web.
  - Rutas, middleware y manejo de peticiones.
  - Uso de plantillas y vistas (ejs, pug, etc.).

## 7. Manipulación de Archivos
- Lectura y escritura de archivos con el módulo `fs`.
- Sincronización vs asincronización en la manipulación de archivos.
- Streams (lectura/escritura continua de datos).

## 8. Bases de Datos
- **Conexión con bases de datos**:
  - MongoDB (usando `mongoose`).
  - MySQL (usando `mysql2` o `sequelize`).
- **Operaciones CRUD** (Crear, Leer, Actualizar, Eliminar).
- **Autenticación y Autorización**:
  - Uso de tokens (JWT).

- **Cluster**: Manejo de múltiples instancias de Node.js en entornos de producción.
