# 2. Instalación y Configuración

## Instalación de Node.js

Para comenzar a usar Node.js, primero es necesario instalarlo en tu sistema. Node.js está disponible para los principales sistemas operativos: **Windows**, **macOS** y **Linux**. A continuación, te explicamos cómo instalar Node.js en cada uno de ellos.

### Instalación en Windows

1. **Descargar el instalador**:
   - Dirígete a la página oficial de Node.js: [https://nodejs.org](https://nodejs.org).
   - Descarga la versión recomendada para la mayoría de los usuarios (LTS) o la versión más reciente (Current).

2. **Ejecutar el instalador**:
   - Una vez descargado el archivo `.msi`, ejecútalo y sigue las instrucciones del asistente de instalación.
   - Durante el proceso, asegúrate de seleccionar la opción de "Add to PATH", lo que permitirá usar `node` y `npm` desde cualquier terminal.

3. **Verificación de la instalación**:
   - Una vez completada la instalación, abre una terminal (cmd o PowerShell) y escribe los siguientes comandos:
     ```bash
     node -v
     npm -v
     ```
   - Estos comandos devolverán las versiones de Node.js y npm instaladas, lo que confirma que la instalación fue exitosa.

### Instalación en macOS

1. **Usar Homebrew** (recomendado):
   - Si tienes **Homebrew** instalado, ejecuta el siguiente comando en la terminal:
     ```bash
     brew install node
     ```
   - Homebrew instalará la última versión de Node.js y npm.

2. **Descargar desde la página oficial**:
   - Si no usas Homebrew, también puedes descargar el instalador `.pkg` desde [https://nodejs.org](https://nodejs.org) y seguir el asistente de instalación.

3. **Verificación de la instalación**:
   - Abre la terminal y ejecuta los siguientes comandos para verificar las versiones de Node.js y npm:
     ```bash
     node -v
     npm -v
     ```

### Instalación en Linux

1. **Instalación usando NVM (Node Version Manager)**:
   - **NVM** es una herramienta que facilita la instalación y gestión de múltiples versiones de Node.js en tu sistema.
   - Para instalar **NVM**, ejecuta el siguiente comando en la terminal:
     ```bash
     curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
     ```
   - Luego, reinicia tu terminal o ejecuta el siguiente comando para cargar NVM:
     ```bash
     source ~/.bashrc
     ```

   - Después de instalar NVM, puedes instalar la última versión de Node.js con:
     ```bash
     nvm install node
     ```

2. **Instalación desde el repositorio oficial**:
   - Si no deseas usar NVM, puedes instalar Node.js directamente desde los repositorios de tu distribución de Linux.
   - Para Ubuntu/Debian, por ejemplo:
     ```bash
     sudo apt update
     sudo apt install nodejs npm
     ```

3. **Verificación de la instalación**:
   - Al igual que en los otros sistemas operativos, verifica la instalación ejecutando:
     ```bash
     node -v
     npm -v
     ```

## Comprobación de la instalación

Una vez que hayas instalado Node.js, es importante verificar que tanto **Node.js** como **npm** están correctamente instalados en tu sistema.

### Comprobación de Node.js

Para verificar que Node.js está instalado correctamente, abre una terminal y ejecuta el siguiente comando:

```bash
node -v
```

Este comando te devolverá la versión de Node.js instalada en tu sistema.

### Comprobación de npm

`npm` (Node Package Manager) es la herramienta de gestión de paquetes para Node.js. Para verificar que npm está correctamente instalado, ejecuta:

```bash
npm -v
```

Este comando mostrará la versión de npm instalada en tu sistema.

## Configuración del entorno de desarrollo

Una vez que Node.js y npm estén instalados, es recomendable configurar tu entorno de desarrollo para trabajar de manera más eficiente. A continuación, algunos pasos importantes para configurar tu entorno.

### Editor de código

Es importante elegir un editor de código adecuado para trabajar con JavaScript y Node.js. Algunos editores populares para trabajar con Node.js son:

- **Visual Studio Code**: Un editor muy popular con soporte integrado para Node.js y una gran cantidad de extensiones.
- **Sublime Text**: Un editor ligero y rápido, compatible con Node.js y extensible con plugins.
- **Atom**: Un editor de código abierto con soporte para JavaScript y Node.js, altamente personalizable.

### Configuración de Variables de Entorno

En algunos proyectos de Node.js, es necesario configurar variables de entorno. Estas variables se utilizan para almacenar configuraciones y credenciales que pueden cambiar según el entorno (desarrollo, producción, etc.).

#### En Linux/macOS

Para agregar variables de entorno, edita tu archivo de configuración de la terminal, como `.bashrc`, `.bash_profile` o `.zshrc`, dependiendo de la shell que uses. Agrega la siguiente línea:

```bash
export NODE_ENV=production
```

Luego, carga las nuevas configuraciones ejecutando:

```bash
source ~/.bashrc   # o el archivo correspondiente
```

#### En Windows

Para configurar variables de entorno en Windows:

1. Ve a "Configuración avanzada del sistema".
2. Haz clic en "Variables de entorno".
3. En "Variables del sistema", haz clic en "Nueva" y agrega la variable `NODE_ENV` con su valor correspondiente (por ejemplo, `production`).

### Inicialización de un Proyecto de Node.js

Ahora que tienes Node.js instalado, puedes crear un proyecto para empezar a trabajar en tus aplicaciones.

1. Crea una carpeta para tu proyecto:
   ```bash
   mkdir mi-proyecto
   cd mi-proyecto
   ```

2. Inicializa el proyecto usando `npm`:
   ```bash
   npm init
   ```
   Este comando te pedirá algunos detalles, como el nombre del proyecto, la versión, la descripción, etc.

3. Si prefieres generar un archivo `package.json` con los valores predeterminados, puedes usar el siguiente comando:
   ```bash
   npm init -y
   ```

Esto creará un archivo `package.json`, que gestionará las dependencias y configuraciones de tu proyecto.


>La instalación y configuración de Node.js es rápida y sencilla en los principales sistemas operativos. Con **Node.js** y **npm** correctamente instalados, puedes empezar a desarrollar aplicaciones del lado del servidor utilizando JavaScript. Además, con herramientas como **NVM** y la gestión de variables de entorno, puedes facilitar el desarrollo y la configuración de tu entorno de manera eficiente.

>Una vez que todo esté instalado y configurado, podrás comenzar a trabajar en tu primer proyecto de Node.js, crear servidores web, interactuar con bases de datos, gestionar rutas de archivos y mucho más.
